## 为什么多个账号容易被平台“盯上”？

现在的平台算法非常敏感，一旦发现多个账号之间有相似的登录信息或网络轨迹，就很容易判定为同一个人操作：

- 同一 IP 地址登录多个账号；
- 登录时间过于接近；
- 浏览器指纹、分辨率、语言设置相似；
- 操作行为雷同（比如频繁发布同类型内容）；

我就曾因为 Instagram 的两个账号使用同一 IP 登录，被直接打上关联标签，限流+功能限制一个星期，几乎崩溃。
